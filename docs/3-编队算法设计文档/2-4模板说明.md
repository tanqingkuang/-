# 二、原则

* 采用策略模式后，对象组不关注子策略的具体实现；因此本文档必须有很强的说明性，让使用者一看就懂。
* 扩展子算法时需要判断是引入新子类还是扩展原子算法，并综合考虑复杂度、扩展性和真实应用场景。
