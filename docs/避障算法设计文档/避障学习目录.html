避障学习目录

这组页面按工程真实调用链讲解避障：配置与 UI 入口、障碍建模、栅格化、A*、工程规则、转向惩罚、去冗余、圆弧与可飞性。每章都尽量用代码片段、图表和小例子对应源码。

章节列表

01 避障总览 从配置和 UI 入口开始，看避障如何一路生成可采用的长机航线。 02 障碍物建模与判障 理解圆形、矩形障碍，膨胀安全距离，以及 `blocked()` 为什么是共享基元。 03 栅格化与坐标转换 把连续的 east/north 坐标变成 A* 可搜索的离散格点。 04 A* 搜索核心 逐步看 open、closed、g、h、f、came_from 如何协作。 05 工程安全规则 端点精确判障、禁止对角切角、网格规模保护等非教材但关键的实现细节。 06 转向惩罚 理解为什么同一个格点从不同方向进入，后续代价可能不同。 07 去冗余与航线转换 把 A* 锯齿格点拉直，并转换为 `WayPointInputS`。 08 圆弧与可飞性 解释转弯半径、腿长约束、圆弧触障和最终 `PlanResult`。
